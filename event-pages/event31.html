About the Workshop

This hands-on workshop focuses on practical system programming topics useful for software developers and engineers. The session includes Linux fundamentals (processes, threads, memory and profiling), essential development tools (GCC, Clang, Makefiles, Git, Valgrind and more), Docker fundamentals (containers, networks, volumes, composition) and scalable system design (synchronous vs asynchronous). The workshop will feature demos, live walkthroughs and Q&A with industry experts from Arista Networks and Persistent Systems.
